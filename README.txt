DEROULEMENT DU JEU:
Dès que le niveau apparaît, le cycliste peut-être contrôlé. Le but du jeu est tout simplement de rejoindre la ligne d’arrivée, matérialisée par un drapeau rouge. Pour que l’arrivée soit comptabilisée, le corps du cycliste doit passer sur le drapeau, le faisant disparaître.

Il meurt dès que son corps entre en contact avec un élément du monde comme : le terrain, une caisse, une planche, une bascule,…

L’enchaînement des niveaux est automatique est se fait après un court délai de 3 secondes.

Les drapeaux jaunes servent à matérialiser la présence d’un “checkpoint”. Si le cycliste chute après avoir passé l’un de ces checkpoints, il recommencera à ce point. (Note : les checkpoints sont considérés dans l’ordre dans lequel il sont validés)

La présence d’un revêtement bleu sur le sol indique une portion de terrain glissante. Sur ce revêtement, les roues n’ont aucune traction.
––––––––––––––––––––––––––
NIVEAUX :
Le jeu compte trois niveau :
1. Hilly : Fait sur la base de l’énoncé.
Permet de se familiariser avec les contrôles et les différents éléments du jeu
2. Roller(coaster) : Un niveau assez dynamique, avec notamment un petit looping
3. Trial : Un niveau beaucoup plus typé parcours d’obstacles

Tous les niveaux sont bien sûr terminables !
––––––––––––––––––––––––––
CONTROLES:

ARROW_UP: Accélérer
ARROW_DOWN : Freiner
ARROW_LEFT / ARROW_RIGHT : Se pencher vers la gauche/la droite

SPACEBAR : Faire demi-tour

SHIFT : Boost

“R” : si mort, appuyer sur la touche “R” pour recommencer le niveau (au dernier checkpoint ou au début du niveau)
